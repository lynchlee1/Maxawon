조건검색 결과 복사

사용 설명서

기본 사용 순서

  1. Chrome 열기를 누릅니다.
  2. 열린 Chrome에서 Cretop에 직접 로그인합니다.
  3. Cretop에서 조건검색을 실행하고 결과 테이블 화면까지 이동합니다.
  4. 앱으로 돌아와 로그인 완료를 누릅니다.
  5. 저장 파일과 최대 페이지를 확인합니다.
  6. 화면 테이블 복사를 누릅니다.

저장 파일

복사 결과는 CSV 파일로 저장됩니다. 다른 위치에 저장하려면 변경 버튼을 사용하세요.

Chrome 종료

실행된 Chrome 종료는 이 앱에서 연 Chrome만 종료합니다. 전체 Chrome 종료는 사용자가 따로 열어 둔 Chrome까지 닫을 수 있습니다.
