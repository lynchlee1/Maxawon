Chrome Session

Chrome 로그인 세션 연결

사용 설명서

  1. Chrome 열기를 누릅니다.
  2. 열린 Chrome에서 Maxawon에 직접 로그인합니다.
  3. Maxawon에서 필요한 화면까지 이동합니다.
  4. 앱으로 돌아와 로그인 완료를 누릅니다.
  5. 왼쪽 메뉴에서 실행할 기능을 선택합니다.

기능 경계

입력
사용자가 Chrome에서 직접 로그인한 브라우저 세션
동작
앱에서 실행할 Chrome을 열고 로그인 완료 상태를 표시합니다.
제외
로그인 자동화, CAPTCHA 처리, 접근 제한 우회는 하지 않습니다.

Chrome 종료 범위

실행된 Chrome 종료
이 앱에서 연 Chrome만 종료합니다.
전체 Chrome 종료
사용자가 따로 열어 둔 Chrome까지 닫을 수 있습니다.

Condition Search Export

조건검색 테이블 CSV 저장

실행 설정

기능 경계

입력
현재 Chrome 화면에 표시된 조건검색 결과 테이블
동작
보이는 결과 테이블을 읽고 지정한 CSV 파일로 저장합니다.
제외
조건검색 실행, 후보 회사 선택, 로그인 자동화는 이 기능에서 처리하지 않습니다.

기능 설명서

  1. Chrome에서 조건검색 결과 테이블 화면을 띄웁니다.
  2. 저장 파일과 최대 페이지를 확인합니다.
  3. 현재 조건검색 테이블 저장을 누릅니다.

App Update

앱 업데이트 확인 및 설치

업데이트 상태

현재 버전
확인 중
업데이트 기준
확인 중
상태
대기 중

기능 경계

입력
현재 설치된 앱 버전과 GitHub 릴리스 업데이트 기준
동작
새 버전을 확인하고, 사용자가 선택하면 다운로드 후 재시작 설치를 진행합니다.
제외
패키징되지 않은 개발 실행 상태에서는 실제 업데이트 설치를 진행하지 않습니다.

기능 설명서

  1. 새 버전 확인을 눌러 릴리스 상태를 확인합니다.
  2. 새 버전이 있으면 업데이트 다운로드를 누릅니다.
  3. 다운로드가 끝나면 재시작 후 설치를 누릅니다.
